How the Game Works

The Pig Dice Game is a simple push-your-luck dice game where players take turns rolling a die to score points, risking it all if they roll a 1. The fundemantals of the game were implemented in the Udemy course: https://www.udemy.com/course/the-complete-javascript-course/

Some Adjustments I made

I created a game, player and dice class to practice object-oriented programming. Additionally, I added multiplayer mode to let the user choose how many players the game should have (2-10 players). To enable multiplayer mode, I needed to change the UI and game logic accordingly.
